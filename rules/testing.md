# Testing Rules

## 19. Minimum Coverage 80%
Không merge nếu dưới 80%.

## 20. Unit Tests Required
Individual functions, utilities, components.

## 21. Integration Tests Required
API endpoints, database operations.

## 22. E2E Tests Required
Playwright, critical user flows.

## 23. TDD MANDATORY
RED (write test) → GREEN (implement) → IMPROVE (refactor).

## 24. Fix Implementation, Not Tests
Trừ khi test sai.

## 25. Testing Agents
tdd-guide (proactive), e2e-runner (Playwright).
