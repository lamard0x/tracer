# Hooks Rules

## 35. PreToolUse: tmux Reminder
Gợi ý tmux cho npm, pnpm, yarn, cargo, etc.

## 36. PreToolUse: git push Review
Mở Zed review trước khi push.

## 37. PreToolUse: doc Blocker
Block tạo file .md/.txt không cần thiết.

## 38. PostToolUse: PR Creation
Log PR URL + GitHub Actions status.

## 39. PostToolUse: Prettier
Auto-format JS/TS sau mỗi edit.

## 40. PostToolUse: TypeScript Check
Run tsc sau edit .ts/.tsx.

## 41. PostToolUse: console.log Warning
Warn nếu có console.log trong edited files.

## 42. Stop: console.log Audit
Check tất cả modified files trước khi kết thúc session.
